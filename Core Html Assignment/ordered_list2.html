Formatting Tags

  1. Bold
    1. Use Case: To make text bold.
    2. Best practices: prefer using strong for semantic importance, and use bold tag for visually appealing purpose.

  2. italic
    1. Use Case: To make text italic.
    2. Best Practices: Prefer using em tag for semantic importance anditalic tag for visually appealing purpose.

  3. Underline
    1. Use Case: To underline text.
    2. Best Practices: Avoid using underlines for non-hyperlinked text as it might be confused with hyperlinks. Consider using CSS for styling.

  4. Strikethrough
    1. Use Case: To strike throught text.
    2. Best Practices: Often used to indicate deleted or no longer relevant content.

  5. Superscript
    1. Use Case: To render text in superscript style.
    2. Best Practices: Commonly used for mathematical expression and annotation.

  6. Subscript
    1. Use Case: To render text in subscript style.
    2. Best Practices: Used for chemical formulas, mathematical expression and annotation.

  7. Marked Text
    1. Use Case: To highlight text for reference or notation
    2. Best Practice: Use it to highlight portions of text for quick identification or reference.

  8. Abbreviation
    1. Use Case: To mark up abbbreviations or acronyms
    2. Best Practise: Provides additional information when the user hover over the abbreviations.
